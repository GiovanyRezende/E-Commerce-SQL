# Projeto Físico de Banco de Dados – E-Commerce

## Descrição
*O projeto se trata de uma modelagem física do [banco de dados de e-commerce.](https://github.com/GiovanyRezende/E-Commerce-Database-Model)* Depois de realizar a modelagem, algumas queries foram requisitadas para responder dúvidas do sistema, sendo elas:

- Quantos pedidos foram feitos por cada cliente?
- Algum vendedor também é fornecedor?
- Qual a relação de produtos, fornecedores e estoques?
- Qual a relação de nomes dos fornecedores e nomes dos produtos?
